# Codevault C

## The Problem
This is a repo for self education and re-education. This is not my first experience with C, but it is my first dedicated deep dive into the language, and into the fundamentals of programming. I am a largely self-taught bootcamp programmer, and after years of JavaScript and Python, I seem to always run into barriers that prevent me from building the stuff I want or need to build. It has become clear that React and Python projects have left me lacking in the fundamentals.

I thought "Okay, well I have the most experience with JavaScript, I will attempt to capitalize on that, and get really good at JavaScript." There may be self-taught bootcamp programmers for whom this works, but I am not one of them. I got to the exercises in Chapter 4 of Eloquent JavaScript and was doing fine until Chapter 4's Linked List exercise. This wasn't something that had ever come up before, and I was unprepared. Many of the resources on Linked Lists that I came across were in C. This was a recurring theme for me: get stuck due to a lack of knowledge on programming fundamentals, go looking for resources, discover that a lot of them are in C. I had also observed that many, if not all, of my colleagues who are doing great professionally, have formal computer science backgrounds where they learned a C-type language.

## The Purpose of this Repo
I have been watching videos on C concepts from the Codevault YouTube channel for a while now, and it has been incredibly useful. I have a project map that is subject to change, but my goals are to:  

- Identify the concepts needed to build a linked list project of my own
- Structure those concepts in the best order possible for learning
- Build my knowledge by completing small projects along with the YouTube videos
- Write a linked list program
  - Be able to explain what I did and why I did it

## Notes
The structures of the project map may change as my journey progresses. Any other projects that get looped into this one will be linked or referenced.
